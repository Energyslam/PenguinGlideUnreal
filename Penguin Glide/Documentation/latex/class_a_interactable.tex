\subsection{Detailed Description}
This class is used for the interactables in the game, namely the power-\/ups. 

The interactables in the game right now is the power-\/up. The power-\/up has two seperate particle system, one to indicate to the player that there is something special about it, and one that plays once the player has picked it up to give some juice to the game. This makes it feel more impactful than simply picking it up. This class is derived from a blueprint. This blueprint holds the static mesh that the interactable has, which is a standard sphere with a material. It creates and attaches the particle system during runtime.
