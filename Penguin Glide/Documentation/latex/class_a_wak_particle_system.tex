\subsection{Detailed Description}
This class has the particle system attached to it to indicate where a wak is. 

This class is used to instantiate an actor in the level on the location of where a wak is to indicate that there is one. It automatically creates a particle system once it is instantiated and attaches it to itself. The wak is not an actual actor but a vertice in the track that has been moved downwards to simulate a pitfall. Due to this, we can\textquotesingle{}t actually attach a particle system to the wak.
